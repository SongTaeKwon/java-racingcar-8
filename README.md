# java-racingcar-precourse

## 요구사항

1. 주어진 횟수 동안 n대의 자동차는 전진 또는 멈출 수 있다.
2. 각 자동차에 이름을 부여할 수 있다. 전진하는 자동차를 출력할 때 자동차 이름을 같이 출력한다.
3. 자동차 이름은 쉼표(,)를 기준으로 구분하며 이름은 5자 이하만 가능하다.
4. 사용자는 몇 번의 이동을 할 것인지를 입력할 수 있어야 한다.
5. 전진하는 조건은 0에서 9 사이에서 무작위 값을 구한 후 무작위 값이 4 이상일 경우이다.
6. 자동차 경주 게임을 완료한 후 누가 우승했는지를 알려준다. 우승자는 한 명 이상일 수 있다.
7. 우승자가 여러 명일 경우 쉼표(,)를 이용하여 구분한다.
8. 사용자가 잘못된 값을 입력할 경우 IllegalArgumentException을 발생시킨 후 애플리케이션은 종료되어야 한다.

## 구현할 기능

### 자동차

1. 자동차 이름으로 자동차를 생성하는 기능
2. 자동차 이름이 5자 이하인지 확인하는 기능
3. 자동차 이름이 비어있지 않은지 확인하는 기능
4. 무작위 값을 생성하는 기능
5. 무작위 값을 기준으로 전진 또는 정지하는 기능
6. 자신의 현재 위치를 출력하는 기능

### 경기

1. 자동차 리스트와 시도 횟수로 경기를 생성하는 기능
2. 자동차 수가 1개 이상인지 확인하는 기능
3. 시도 횟수가 1번 이상인지 확인하는 기능
4. 이동 시도를 1번 하는 기능
5. 이동 시도를 시도 횟수만큼 하는 기능
6. 경기에 속한 자동차들에게 자신의 현재 위치를 출력하도록 하는 기능
7. 경기의 우승자를 출력하는 기능

### 기타

1. 시도 횟수를 정수로 바꾸는 기능
2. 자동차 문자열을 자동차 리스트로 만드는 기능
